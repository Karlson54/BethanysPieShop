# Bethany's Pie Shop

The "Bethany's Pie Shop" project is a web application designed for selling pies online. It is built using ASP.NET Core MVC, with a frontend enhanced by Bootstrap and jQuery for a responsive and engaging user experience.

### Key Features

1. **Pies of the Week**: The homepage highlights a selection of featured pies. Users can click on a pie to view detailed descriptions and add items to their cart.
2. **Pie Categories**: Users can browse pies by categories via a drop-down menu.
3. **Search Functionality**: A search bar allows users to find pies by keywords.
4. **User Authentication**: Registration and login pages enable users to create and access accounts.
5. **Shopping Cart**: Users can add pies to their cart and proceed to checkout, with validation for authorized users.

### Technologies Used

- **Backend**: ASP.NET Core MVC and ASP.NET API.
- **Frontend**: Bootstrap and jQuery.
- **Database**: Entity Framework Core for database management.
- **Authentication**: Built-in ASP.NET Core Identity system.

### Installation and Usage

To run the application locally:

1. Clone the repository.
2. Navigate to the project directory.
3. Restore NuGet packages.
4. Update the database.
5. Run the application using `dotnet run`.
6. Access the app via `http://localhost:5000`.
